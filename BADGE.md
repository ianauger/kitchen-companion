## CI/CD Badge

Add this to your README.md:

```markdown
[![Build and Deploy](https://github.com/${{ github.repository }}/actions/workflows/build-and-deploy.yml/badge.svg)](https://github.com/${{ github.repository }}/actions/workflows/build-and-deploy.yml)
```

Or with the actual repo path (replace `OWNER` with your GitHub username/org):

```markdown
[![Build and Deploy](https://github.com/OWNER/kitchen-companion-app/actions/workflows/build-and-deploy.yml/badge.svg)](https://github.com/OWNER/kitchen-companion-app/actions/workflows/build-and-deploy.yml)
```
