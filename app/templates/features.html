Features

Everything you need to manage your personal digital cookbook.

📚

Recipe Storage

Store your favorite recipes with detailed information including cooking instructions, prep times, serving sizes, and difficulty levels. Never lose a beloved family recipe again.

🖼️

Automatic Image Handling

Provide an image URL when creating or updating a recipe, and Kitchen Companion automatically downloads and stores it locally. Your cookbook stays fast and self-contained with no external dependencies for images.

POST /api/recipes { "image_url": "https://...", ... } → image stored locally

🏷️

Smart Tagging System

Organize your recipes with a flexible tagging system. Tags are categorized by type for intelligent filtering:

🍝 Cuisine
🥩 Protein
🌶️ Spice Level
🧄 Ingredient
✨ Custom
🔍

Powerful Search & Filter

Find exactly what you're craving. Search by recipe name and filter by cuisine type, protein, spice level, or any combination of tags. Perfect for meal planning or using up ingredients you have on hand.

🎲

Random Discovery

Not sure what to cook? The home page shows a random selection of 6 recipes to spark inspiration. Refresh for a new set whenever you need culinary ideas.

⚡

Full REST API

Access all features programmatically through a clean REST API. Perfect for integrations, scripts, or building your own frontend.

GET POST PUT DELETE
🎨

Modern Responsive UI

Built with Tailwind CSS for a beautiful, responsive interface that works perfectly on desktop, tablet, and mobile. Clean sidebar navigation keeps everything within easy reach.
