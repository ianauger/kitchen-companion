API Reference

Complete documentation for the Kitchen Companion REST API endpoints.

Base URL

http://localhost:5000/api
GET /recipes

Retrieve all recipes with optional filtering.

Query Parameters

Parameter Type Description
tag string Filter by tag name
tag_type string Filter by tag type (cuisine, protein, spice_level, ingredient, custom)
difficulty string Filter by difficulty (easy, medium, hard)
search string Search recipes by title (case-insensitive)

Example Response

[
  {
    "id": 1,
    "title": "Spicy Thai Basil Chicken",
    "source_url": null,
    "image_url": "https://example.com/image.jpg",
    "image_path": "uploads/recipes/recipe_1_xxx.jpg",
    "instructions": "...",
    "cooking_time": 20,
    "prep_time": 10,
    "servings": 4,
    "difficulty": "medium",
    "tags": [
      {"id": 1, "name": "Thai", "tag_type": "cuisine"}
    ]
  }
]
POST /recipes

Create a new recipe. Optionally provide image_url to have the image downloaded and stored locally.

Request Body

{
  "title": "Spicy Thai Basil Chicken",
  "instructions": "1. Heat oil in a wok...\n2. Add garlic and chili...",
  "source_url": "https://example.com/recipe",
  "image_url": "https://example.com/cover.jpg",
  "cooking_time": 20,
  "prep_time": 10,
  "servings": 4,
  "difficulty": "medium",
  "tags": [
    {"name": "Thai", "tag_type": "cuisine"},
    {"name": "chicken", "tag_type": "protein"},
    {"name": "spicy", "tag_type": "spice_level"}
  ]
}

Required Fields

  • title — Recipe name
  • instructions — Cooking instructions

Response

Returns the created recipe object with 201 Created status. The image_path field will contain the local path if an image_url was provided.

GET /recipes/<id>

Retrieve a specific recipe by ID.

Returns a single recipe object or 404 Not Found if the recipe doesn't exist.

PUT /recipes/<id>

Update an existing recipe. If image_url is provided and differs from the current value, the new image will be downloaded and the old local image deleted.

Include only the fields you want to update in the request body.

DELETE /recipes/<id>

Delete a recipe and its associated local image file.

Returns 204 No Content on success. Also deletes the locally stored cover image if one exists.

GET /recipes/random

Get a random selection of recipes.

Query Parameters

count — Number of recipes to return (default: 6, max: 20)

GET /tags

Retrieve all tags with optional filtering.

Use ?tag_type=cuisine to filter by tag type.

Tag Types

  • cuisine — Italian, Thai, Indian, etc.
  • protein — chicken, beef, tofu, etc.
  • spice_level — mild, medium, hot
  • ingredient — specific ingredient tags
  • custom — user-defined tags
