# Kitchen Companion v1 Prototype

This directory contains the original Phase 1 prototype from which `kitchen-companion-app` was built.

- **`index.html`** — Single-page recipe viewer
- **`script.js`** — Frontend logic
- **`styles.css`** — Styling
- **`server.py`** — Simple Python HTTP server
- **`sources.json`** — Recipe source URLs
- **`phase2-spec.md`** — Bridge spec from Phase 1 → Phase 2
- **`recipes/`** — Original JSON recipe database

Kept for reference. The active project is the parent `kitchen-companion-app/`.
